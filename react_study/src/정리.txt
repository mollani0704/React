React 란?
리액트는 페이스북사에서 만든 자바스크립트 라이브러리입니다.
페이스북에서 자체적으로 이용되고 있을 뿐 아니라 
많은 웹 어플리케이션이 리액트로 만들어졌습니다.

"리액트는 유저 인터페이스를 만들 수 있는 라이브러리입니다." ->SPA
-> 사용자에게 보여지는 UI 만들고 사용자가 클릭을 하거나 이벤트가 발생하게 되면 
   그 이벤트에 맞게 반응하도록 만들어진 라이브러리입니다.

웹개발  //   앱개발(우리)   

복잡한 웹 어플리케이션을 만들 때 MVC 패턴을 사용합니다.

  Model           View        Controller
             (React가 담당)

* 리액트는 사용자에게 UI를 보여주고 이벤트를 처리하는 일들을 할 수 있습니다.(데이터)
React : 웹 어플리케이션 UI 개발
React Native : 모바일 어플리케이션
React + Electron : 데스크탑 어플리케이션

리액트는 컴포넌트로 이루어져 있는 UI 라이브러리입니다"
컴포넌트 : 버튼과 같은 하나, 한 가지의 기능을 수행하는 UI 단위를 말합니다. 
          서로 독립적이고 재사용이 가능한 것을 의미합니다.

[JSX]리액트 확장자
리액트와 마찬가지로 페이스북에서 개발한 확장 구문입니다. 
- 태그는 꼭 닫혀야합니다.
- 두개 이상의 태그는 무조건 하나의 태그로 감싸야 합니다.
  불필요한 div를 사용하고 싶지 않다면 <> 사용하면 됩니다.
- <div></div>와 <></>의 차이점
   <div></div>는 스타일시트를 적용
   <></>는 단순히 묶음(그룹)기능
- 주석은 {/*  */} 표현합니다.   
- CSS 속성은 스네이크 표기법 -> 카멜표기법
  background-color -> backgroundColor
