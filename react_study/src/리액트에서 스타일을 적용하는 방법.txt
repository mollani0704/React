1.컴포넌트에 직접스타일 적용하면 적용이 안됩니다
2.컴포넌트 파일안으로 직접 이동을 해서 스타일을 선언할 수 있다
3.스타일의 선택자명을 중복해서 적용이 되므로 중복이 되지않게 적용을 할려면
  각각 컴포넌트에 적용 스타일의 파일을 파일명.module.css로 저장을 하고
  적용시는 컴포넌트 상단에 import styles from './파일명.module.css'; 연결 설정하고
  해당되는 요소에는 className={styles.Box}라고 적용

  CSS Module
  -리액트 프로젝트에서 컴포넌트를 스타일링 할 때 
  CSS Module 기술을 사용하면 CSS 클래스가 중첩되는 것을 완벽히 방지할 수 있습니다.
  -* CSS 파일 확장자를 .module.css로 해야함
  -  파일경로, 파일이름, 클래스이름, 해쉬값등 사용

  styled-components
  JS안에 CSS를 작성하는 라이브러리입니다. 
  yarn add styled-components

  react-icons
  (https://react-icons.github.io/react-icons/#/)
  yarn add react-icons
